# 软件使用说明书

## 1. 概述

NoStockHub 是一款基于 Web 的软件项目管理平台，支持多项目管理、任务跟踪、敏捷看板、甘特图、冲刺管理、工时记录、文件管理和报表生成。系统采用前后端分离架构，前端基于 Vue 3 + Element Plus，后端基于 Django REST Framework。

## 2. 快速开始

### 2.1 注册账号

1. 打开系统首页，点击登录页底部的 **"立即注册"** 链接
2. 填写用户名（3-20 个字符）、邮箱、密码（至少 6 位）、确认密码，手机号可选
3. 点击 **"注册"** 按钮，注册成功后自动返回登录页

### 2.2 登录

1. 输入**用户名或邮箱**（支持两种方式登录）
2. 输入密码
3. 可选勾选"记住我"，下次访问自动填充用户名
4. 点击 **"登录"** 按钮进入系统

### 2.3 界面布局

登录后进入系统主界面，包含以下区域：

- **顶部导航栏**：系统名称、通知图标、用户头像下拉菜单
- **左侧菜单栏**：仪表盘、项目列表，展开项目后可进入各功能页面
- **主内容区**：当前功能页面的操作区域

## 3. 仪表盘

仪表盘是登录后的默认首页，展示个人工作概览：

- **统计卡片**：参与项目数、待办任务、进行中任务、已完成任务
- **我的任务**：当前用户负责的任务列表，点击可查看详情
- **我的项目**：已参与的最近项目，点击进入项目详情
- **最近活动**：项目内的操作动态时间线

## 4. 项目管理

### 4.1 项目列表

点击左侧 **"项目列表"** 进入，展示所有可见项目：

- **搜索**：按项目名称搜索
- **状态筛选**：按规划中 / 进行中 / 已完成 / 已归档筛选
- **卡片展示**：项目名称、进度、任务统计、状态、可见性标签
- **新建项目**：点击右上角 **"+ 新建项目"** 按钮，填写项目名称、描述、起止日期、可见性（公开/私密）

### 4.2 项目概览

进入项目后，默认显示**项目概览**页面：

- 项目基本信息（描述、日期、负责人）
- 团队成员列表
- 任务统计（总数 / 已完成 / 进行中）
- 进度燃尽图
- 最近活动时间线

### 4.3 项目详情

点击项目内的 **"项目详情"** 可进行更多操作：

- **编辑项目**：修改名称、描述、日期、状态、可见性
- **团队成员**：查看成员列表，点击 **"+ 邀请成员"** 搜索用户名并选择角色（成员/观察者）
- **里程碑**：创建、编辑、切换完成状态、删除里程碑
- **进入看板**：快捷跳转到任务看板
- **删除项目**：删除整个项目及其所有数据（谨慎操作）

### 4.4 项目状态

| 状态 | 含义 |
|------|------|
| 规划中 | 项目仍在规划设计阶段 |
| 进行中 | 项目正在执行 |
| 已完成 | 项目已交付完成 |
| 已归档 | 项目已归档，不再活跃 |

## 5. 任务管理

### 5.1 任务列表

左侧选择项目后，点击 **"任务列表"** 进入：

- **筛选工具栏**：
  - 搜索框：按任务名称或描述搜索（自动防抖搜索）
  - 状态多选：待办 / 进行中 / 审核中 / 已完成 / 阻塞
  - 优先级：低 / 中 / 高 / 紧急
  - 类型：任务 / 缺陷 / 史诗
  - 负责人：按团队成员筛选
  - **"清除筛选"** 按钮：一键重置所有筛选条件
- **表格列表**：
  - 左侧复选框：勾选即可将任务标记为完成
  - 点击任务行：打开任务详情对话框
  - 任务名称、类型标签、优先级标签、负责人头像、截止日期（过期显示红色）、状态下拉
  - 排序：点击列头切换升降序
- **编辑/删除**：每行操作列有编辑和删除按钮
- **分页**：底部页码导航，可调整每页显示条数
- **新建任务**：点击右上角 **"+ 新任务"** 按钮

### 5.2 创建/编辑任务

打开任务对话框后，填写以下信息：

| 字段 | 说明 | 必填 |
|------|------|------|
| 标题 | 任务名称 | 是 |
| 描述 | 详细说明 | 否 |
| 类型 | 任务 / 缺陷 / 史诗 | 否 |
| 优先级 | 低 / 中 / 高 / 紧急 | 否 |
| 状态 | 当前任务状态 | 否 |
| 开始日期 | 计划开始时间 | 否 |
| 截止日期 | 计划完成时间 | 否 |
| 预估工时 | 预计耗时（小时） | 否 |
| 负责人 | 从项目成员中选择 | 否 |
| 冲刺 | 所属冲刺（可选） | 否 |
| 父任务 | 上级任务（创建子任务） | 否 |

### 5.3 任务看板

点击 **"任务看板"** 进入敏捷看板视图：

- **选择看板**：下拉切换已有看板，或新建看板
- **默认列**：待办 → 进行中 → 审核中 → 已完成
- **拖拽操作**：将任务卡片拖拽到其他列，自动更新任务状态
- **WIP 限制**：列头部显示任务计数，超出限制时显示红色警告
- **列管理**：编辑列名称、设置 WIP 上限、删除列
- **快速创建**：列底部的 **"+ 添加任务"** 按钮，在指定列下创建任务
- **任务卡片**：显示优先级、截止日期、工时、标题、负责人头像
- **查看详情**：点击卡片打开右侧抽屉，显示任务完整信息并可编辑/删除

## 6. 甘特图

点击 **"甘特图"** 进入项目时间线视图：

- **视图切换**：日 / 周 / 月三种视图模式
- **导航**：左右箭头翻页，点击"今天"回到当前日期
- **时间轴**：底部滑块可缩放和滚动时间范围
- **任务条**：半透明背景 + 进度填充，颜色表示状态
- **里程碑**：绿色菱形标记在截止日期处
- **依赖连线**：虚线箭头展示任务间的前驱/后继关系
- **筛选**：可选择显示/隐藏里程碑和依赖关系
- **今天标记**：红色竖线标识当前日期
- **图例**：底部显示各元素含义

## 7. 冲刺管理

点击 **"冲刺管理"** 进入 Scrum 冲刺页面：

- **活跃冲刺**：当前正在进行的冲刺，显示：
  - 冲刺名称、状态、目标、日期
  - 进度条
  - 已添加的任务标签（可移除）
  - 添加任务下拉框（从项目任务中选择）
  - **查看燃尽图**：弹出燃尽图对话框
  - **完成冲刺**：将冲刺标记为已完成
- **规划中/已完成冲刺**：显示历史冲刺列表，可启动或删除
- **新建冲刺**：填写名称、目标、起止日期

### 7.1 燃尽图

- 灰色虚线：理想进度线（均匀递减）
- 蓝色实线：实际剩余任务数
- 纵轴：剩余任务数 / 横轴：日期
- 鼠标悬停查看每日数据

## 8. 工时管理

点击 **"工时管理"** 进入工作日志页面：

- **记录列表**：显示任务、用户、工时、日期、描述
- **新建工时**：选择项目任务、填写工时（小时）、日期、描述
- **删除工时**：可删除自己记录的工时

### 8.1 费率管理

点击 **"费率管理"** 设置成员工时费率：

- 选择用户、设置时薪（元/小时）、生效日期
- 用于后续成本核算

## 9. 文件管理

点击 **"文件管理"** 进入项目文件页面：

- **上传文件**：支持图片、文档、压缩包、代码文件等多种格式（最大 50MB）
- **文件列表**：显示文件名、大小、类型、上传时间
- **下载**：点击下载按钮保存文件到本地
- **删除**：仅上传者可以删除自己上传的文件
- **共享**：同项目成员均可查看和下载

## 10. 报表

点击 **"报表"** 进入项目数据分析页面：

- **日期筛选**：选择起止日期范围，提供本周/本月/上个月快捷选项
- **四个分析维度**：

### 10.1 概览
- 统计卡片：任务总数、已完成、总工时、完成率
- 项目进度环形图
- 任务状态分布饼图（待办/进行中/审核中/已完成/阻塞）
- 任务优先级分布饼图

### 10.2 任务统计
- 每日完成任务累计趋势（面积图）
- 新增 vs 完成任务对比
- 按负责人任务分布（柱状图）
- 按类型任务分布（柱状图）

### 10.3 工时分析
- 总工时、日均工时、可计费工时、活跃成员数
- 每日工时趋势图
- 成员工时分布柱状图

### 10.4 燃尽图
- 选择冲刺查看燃尽图
- 理想剩余线 vs 实际剩余线

### 10.5 导出报表
1. 点击 **"导出报表"** 按钮
2. 输入报表名称（默认格式：`概览报表_2026-05-29`）
3. 选择导出格式（CSV / Excel / PDF）
4. 点击 **"生成报表"**，系统自动生成并下载
5. 历史报表可在底部表格中查看和重新下载

## 11. 系统设置

点击右上角头像 → **"设置"**，或通过左侧菜单进入：

### 11.1 个人信息
- **头像**：点击上传新头像（支持 JPG/PNG/GIF 等格式）
- **用户名**：显示当前用户名
- **邮箱**：绑定邮箱地址
- **修改密码**：输入当前密码和新密码

### 11.2 外观设置
- **主题切换**：明亮模式 / 暗黑模式，即时生效
- **语言**：界面语言切换
- **字体大小**：调整全局字体大小，即时预览

### 11.3 安全设置
- **双因素认证**：开启/关闭手机或邮箱验证（可选）

## 12. 常见问题

**Q：忘记密码怎么办？**
A：请联系项目管理员或开发者重置密码。

**Q：看板中任务卡片信息不完整？**
A：任务卡片显示优先级、截止日期、工时、标题和负责人。点击卡片可查看完整详情。

**Q：报表中看不到任务数据？**
A：检查日期范围是否覆盖了任务的创建/截止日期。可扩大日期范围或点击快捷选项"本月"重试。

**Q：文件上传后刷新消失了？**
A：请确保在上传文件时已进入具体的项目页面（URL 中包含项目 ID），而非全局上传。

**Q：甘特图中看不到依赖连线？**
A：需先在任务间创建依赖关系。在任务详情中可为任务添加前驱/后继依赖。

**Q：导出的报表没有内容？**
A：检查当前 Tab 页是否有数据。报表导出的是当前显示的分析维度数据。
