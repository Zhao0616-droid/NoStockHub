软件工程理论与实践课程设计项目

## 项目信息
- 项目名称：软件项目管理平台
- 项目描述：基于Web的项目管理系统，支持多项目管理、甘特图、任务跟踪、看板、团队协作。
- 开发模式：前后端分离
- 技术栈：前端Vue/React；后端SpringBoot/Django；数据库MySQL
